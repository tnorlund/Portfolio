# Scripts Directory

This directory contains various development and maintenance scripts for the Portfolio project.

## PR Review Tools

**Note:** PR review scripts have been moved to `.github/pr-reviews/`. See [PR Review Tools README](../.github/pr-reviews/README.md) for documentation on fetching and analyzing PR comments from CodeRabbit.

## Other Scripts

For documentation on other scripts in this directory, see individual script files or use `--help` flags where available.

Common script categories:
- **Agent scripts**: `agent_*.py` - Training and monitoring for ML agents
- **Dev scripts**: `dev*.py` - Development utilities and debugging tools
- **Export scripts**: `export_*.py` - Data export utilities
- **Deployment scripts**: `*.sh` - Deployment and CI/CD helpers
- **Test scripts**: `test_*.py`, `*_test.sh` - Testing utilities
